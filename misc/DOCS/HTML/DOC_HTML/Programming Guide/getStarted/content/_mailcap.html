Platform-Specific Details: Changing the Mailcap Specification of "plain/text"

UNIX:

Generally, per-user configuration of mailcap is done in a file named ~/.mailcap. If you have no such file, contact your system administrator to find out where your mailcap information is.

  1. In your mailcap file, find the line beginning with "text/plain". For example: 
    text/plain; /usr/openwin/bin/textedit %s
  2. Comment out the line (using the '#' character) or delete it. For example: 
    #text/plain; /usr/openwin/bin/textedit %s
